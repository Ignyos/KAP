## UI Design (Minimal)

Scope for now: Main page only.

### Main Page Structure

1. Sticky header at the top that always stays visible.
2. Header text: Kitchen & Pantry
3. Directly under header: horizontal tab row with three tabs:
	 - Lists
	 - Templates
	 - Recipes
4. Content area under tabs updates based on selected tab.

### Selected Tab Content

When a tab is selected, the top row of the content area shows:

- Left side: subheading
- Right side: create button

Expected labels by tab:

- Lists tab:
	- Subheading: My Lists
	- Button: + New List
- Templates tab:
	- Subheading: My Templates
	- Button: + New Template
- Recipes tab:
	- Subheading: My Recipes
	- Button: + New Recipe
